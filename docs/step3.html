EnemyController.cs

敵キャラクターの動きを制御するスクリプトです。

設定できる値（Inspector）

moveSpeed

移動の速さ（初期値: 2）

patrolDistance

パトロール範囲（初期値: 3）

moveRight

最初に右に進むか

wallCheckDistance

壁検知の距離（初期値: 0.5）

1. パトロール移動

開始位置から一定距離を往復します。

private void Patrol()
{
    // 開始位置からの距離を計算
    float distanceFromStart = transform.position.x - startPosition.x;

    // パトロール範囲を超えたら反転
    if (distanceFromStart > patrolDistance)
    {
        currentDirection = -1f;  // 左に向きを変える
    }
    else if (distanceFromStart < -patrolDistance)
    {
        currentDirection = 1f;   // 右に向きを変える
    }
}

2. 壁の検知

Raycastで前方に壁があるかチェックします。

private void CheckWall()
{
    // 壁チェック用の位置を計算
    Vector2 checkPos = (Vector2)transform.position +
                       Vector2.right * currentDirection * 0.5f;

    // 前方に向けてレイを飛ばす
    RaycastHit2D hit = Physics2D.Raycast(
        checkPos,
        Vector2.right * currentDirection,
        wallCheckDistance,
        groundLayer
    );

    // 何かに当たったら反転
    if (hit.collider != null)
    {
        currentDirection *= -1f;
    }
}

Raycast: 見えない「光線」を飛ばして、何かに当たるかチェックします。壁や崖の検出によく使います。

3. 移動の適用

private void Move()
{
    // 速度を設定
    rb.linearVelocity = new Vector2(currentDirection * moveSpeed, rb.linearVelocity.y);

    // スプライトの向きを変更
    if (spriteRenderer != null)
    {
        spriteRenderer.flipX = currentDirection < 0;
    }
}

ItemController.cs

アイテムの動きとアニメーションを制御するスクリプトです。

1. 浮遊アニメーション

Sin関数を使って上下に滑らかに動かします。

private void FloatAnimation()
{
    // Sin関数で-1〜1の値を得る
    // 時間 × 速度で波の進み具合を決める
    // 振幅（floatAmount）を掛けて動く幅を決める
    float newY = startPosition.y +
                 Mathf.Sin(elapsedTime * floatSpeed) * floatAmount;

    transform.position = new Vector3(
        transform.position.x,
        newY,
        transform.position.z
    );
}

Sin波で滑らかな上下運動

Mathf.Sin: 0から始まり、1まで上がって、-1まで下がって、また0に戻る...を繰り返す関数です。滑らかな繰り返しアニメーションに最適！

2. Triggerで取得処理

ColliderをTriggerにすると、すり抜けつつ衝突を検知できます。

void Start()
{
    // ColliderをTriggerに設定
    BoxCollider2D collider = GetComponent<BoxCollider2D>();
    if (collider != null)
    {
        collider.isTrigger = true;
    }

    // タグを設定
    gameObject.tag = "Item";
}

// Triggerに何かが入ってきた時
private void OnTriggerEnter2D(Collider2D other)
{
    // プレイヤーかチェック
    if (other.gameObject.CompareTag("Player"))
    {
        // GameManagerに通知
        if (GameManager.Instance != null)
        {
            GameManager.Instance.CollectItem();
        }
        // アイテムを消す
        Destroy(gameObject);
    }
}

ColliderとTriggerの違い

Collider（isTrigger = false）

  • ・物理的にぶつかる
  • ・押し合いが発生する
  • ・OnCollisionEnter2D で検知
  • ・例: 地面、壁、敵

Trigger（isTrigger = true）

  • ・すり抜ける
  • ・触れたことだけ検知
  • ・OnTriggerEnter2D で検知
  • ・例: アイテム、ゴール、ダメージエリア
